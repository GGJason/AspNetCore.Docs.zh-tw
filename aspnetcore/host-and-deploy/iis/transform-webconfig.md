## <a name="build-configuration"></a><span data-ttu-id="cb839-113">建置組態</span><span class="sxs-lookup"><span data-stu-id="cb839-113">Build configuration</span></span>

<span data-ttu-id="cb839-114">組建組態會第一個執行。</span><span class="sxs-lookup"><span data-stu-id="cb839-114">Build configuration transforms are run first.</span></span>

<span data-ttu-id="cb839-115">請為每個需要進行 *web.config* 轉換的 [組建組態 (Debug|Release)](/dotnet/core/tools/dotnet-publish#options) 包含一個 *web.{CONFIGURATION}.config* 檔案。</span><span class="sxs-lookup"><span data-stu-id="cb839-115">Include a *web.{CONFIGURATION}.config* file for each [build configuration (Debug|Release)](/dotnet/core/tools/dotnet-publish#options) requiring a *web.config* transformation.</span></span>

<span data-ttu-id="cb839-116">在下列範例中，會在 *web.Release.config* 中設定一個組態特定的環境變數：</span><span class="sxs-lookup"><span data-stu-id="cb839-116">In the following example, a configuration-specific environment variable is set in *web.Release.config* :</span></span>

```xml
<?xml version="1.0"?>

  <location>
    <system.webServer>
      <aspNetCore>
        <environmentVariables xdt:Transform="InsertIfMissing">
          <environmentVariable name="Configuration_Specific" 
                               value="Configuration_Specific_Value" 
                               xdt:Locator="Match(name)" 
                               xdt:Transform="InsertIfMissing" />
        </environmentVariables>
      </aspNetCore>
    </system.webServer>
  </location>
</configuration>
```

<span data-ttu-id="cb839-117">當組態設定為 *Release* 時，就會套用轉換：</span><span class="sxs-lookup"><span data-stu-id="cb839-117">The transform is applied when the configuration is set to *Release* :</span></span>

<span data-ttu-id="cb839-118">組態的 MSBuild 屬性為 `$(Configuration)`。</span><span class="sxs-lookup"><span data-stu-id="cb839-118">The MSBuild property for the configuration is `$(Configuration)`.</span></span>
