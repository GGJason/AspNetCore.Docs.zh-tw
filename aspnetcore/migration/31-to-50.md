# <a name="migrate-from-aspnet-core-31-to-50"></a>從 ASP.NET 核心 3.1 移至 5.0

作者：[Scott Addie](https://github.com/scottaddie)

本文介紹如何將現有ASP.NET Core 3.1 專案更新為 ASP.NET Core 5.0。

> [!IMPORTANT]
> ASP.NET核心 5.0 當前處於預覽狀態。

## <a name="update-net-core-sdk-version-in-globaljson"></a>更新 global.json 中的 .NET Core SDK 版本

如果您依賴[global.json](/dotnet/core/tools/global-json)檔來定位特定的 .NET Core`version`SDK 版本, 請將屬性更新為已安裝的 .NET 5.0 SDK 版本。 例如：

```diff
{
  "sdk": {
-    "version": "3.1.200"
+    "version": "5.0.100-preview.2.20176.6"
  }
}
```

## <a name="update-the-target-framework"></a>更新目標框架

在專案檔案中,將[目標框架 Moniker (TFM)](/dotnet/standard/frameworks)`netcoreapp5.0`更新為:

```diff
<Project Sdk="Microsoft.NET.Sdk.Web">

  <PropertyGroup>
-    <TargetFramework>netcoreapp3.1</TargetFramework>
+    <TargetFramework>netcoreapp5.0</TargetFramework>
  </PropertyGroup>

</Project>
```

## <a name="update-package-references"></a>更新套件參考

在專案檔中,將每個`Microsoft.AspNetCore.*`和`Microsoft.Extensions.*`包引用的屬性`Version`更新為 5.0.0 或更高版本。 例如：

```diff
<ItemGroup>
-    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="3.1.2" />
-    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="3.1.2" />
+    <PackageReference Include="Microsoft.AspNetCore.Mvc.NewtonsoftJson" Version="5.0.0-preview.2.20167.3" />
+    <PackageReference Include="Microsoft.Extensions.Caching.Abstractions" Version="5.0.0-preview.2.20160.3" />
</ItemGroup>
```

## <a name="update-docker-images"></a>更新 Docker 映像檔

對於使用 Docker 的應用,請使用包含 ASP.NET 核心 5.0 的基本映射。 例如：

```bash
docker pull mcr.microsoft.com/dotnet/core/aspnet:5.0
```

## <a name="review-breaking-changes"></a>檢視變更

有關從 .NET Core 3.1 到 .NET 5.0 的中斷更改,請參閱[從版本 3.1 遷移到 5.0 的中斷更改](/dotnet/core/compatibility/3.1-5.0)。 ASP.NET核心和實體框架核心也包含在清單中。
