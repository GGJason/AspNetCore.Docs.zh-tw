<span data-ttu-id="0093b-163">如果您的應用程式在類別中記錄提供者初始化、篩選和設定載入 `Startup` ，請將該程式碼移至 `Program.Main` ：</span><span class="sxs-lookup"><span data-stu-id="0093b-163">If your app does logging provider initialization, filtering, and configuration loading in the `Startup` class, move that code to `Program.Main`:</span></span>

* <span data-ttu-id="0093b-164">提供者初始化：</span><span class="sxs-lookup"><span data-stu-id="0093b-164">Provider initialization:</span></span>

  <span data-ttu-id="0093b-165">1.x 範例：</span><span class="sxs-lookup"><span data-stu-id="0093b-165">1.x example:</span></span>

  ```csharp
  public void Configure(IApplicationBuilder app, ILoggerFactory loggerFactory)
  {
      loggerFactory.AddConsole();
  }
  ```

  <span data-ttu-id="0093b-166">2.2 範例：</span><span class="sxs-lookup"><span data-stu-id="0093b-166">2.2 example:</span></span>

  ```csharp

  public static void Main(string[] args)

      var webHost = new WebHostBuilder()
          // ...
          .ConfigureLogging((hostingContext, logging) =>
          {
              logging.AddConsole();
          })
          // ...
  }
  ```

* <span data-ttu-id="0093b-167">篩選：</span><span class="sxs-lookup"><span data-stu-id="0093b-167">Filtering:</span></span>

  <span data-ttu-id="0093b-168">1.x 範例：</span><span class="sxs-lookup"><span data-stu-id="0093b-168">1.x example:</span></span>

  ```csharp
  public void Configure(IApplicationBuilder app, ILoggerFactory loggerFactory)
  {
      loggerFactory.AddConsole(LogLevel.Information);
      // or
      loggerFactory.AddConsole((category, level) => 
          category == "A" || level == LogLevel.Critical);
  }
  ```

  <span data-ttu-id="0093b-169">2.2 範例：</span><span class="sxs-lookup"><span data-stu-id="0093b-169">2.2 example:</span></span>

  ```csharp
  public static void Main(string[] args)
  {
      var webHost = new WebHostBuilder()
          // ...
          .ConfigureLogging((hostingContext, logging) =>
          {
              logging.AddConsole()
                     .AddFilter<ConsoleLoggerProvider>
                         (category: null, level: LogLevel.Information)
                     // or
                     .AddFilter<ConsoleLoggerProvider>
                         ((category, level) => category == "A" ||
                             level == LogLevel.Critical)
              );
          })
          // ...
  }
  ```

* <span data-ttu-id="0093b-170">設定載入：</span><span class="sxs-lookup"><span data-stu-id="0093b-170">Configuration loading:</span></span>

  <span data-ttu-id="0093b-171">1.x 範例：</span><span class="sxs-lookup"><span data-stu-id="0093b-171">1.x example:</span></span>

  ```csharp
  public void Configure(IApplicationBuilder app, ILoggerFactory loggerFactory)
  {
      loggerFactory.AddConsole(Configuration);
  }
  ```

  <span data-ttu-id="0093b-172">2.2 範例：</span><span class="sxs-lookup"><span data-stu-id="0093b-172">2.2 example:</span></span>

  ```csharp

  {
      var webHost = new WebHostBuilder()
          // ...
          .ConfigureLogging((hostingContext, logging) =>

              logging.AddConfiguration(hostingContext.Configuration.GetSection("Logging"));
              logging.AddConsole();
          })
          // ...
  }
  ```
