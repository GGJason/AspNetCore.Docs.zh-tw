## <a name="call-configurekestrel-instead-of-usekestrel"></a><span data-ttu-id="036e0-121">呼叫而不是 UseKestrel ConfigureKestrel</span><span class="sxs-lookup"><span data-stu-id="036e0-121">Call ConfigureKestrel instead of UseKestrel</span></span>

```csharp

        .UseStartup<Startup>()
        .ConfigureKestrel((context, options) =>
        {
            // Set properties and call methods on options
        });
```
