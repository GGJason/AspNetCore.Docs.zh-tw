## <a name="update-target-framework-moniker-tfm"></a><span data-ttu-id="b03f6-106">更新 Target Framework Moniker (TFM)</span><span class="sxs-lookup"><span data-stu-id="b03f6-106">Update Target Framework Moniker (TFM)</span></span>

<span data-ttu-id="b03f6-109">以.NET Framework 為目標的專案可以繼續使用版本大於或等於.NET Framework 4.6.1 的 TFM:</span><span class="sxs-lookup"><span data-stu-id="b03f6-109">Projects targeting .NET Framework may continue to use the TFM of a version greater than or equal to .NET Framework 4.6.1:</span></span>

## <a name="update-net-core-sdk-version-in-globaljson"></a><span data-ttu-id="b03f6-110">更新 global.json 中的 .NET Core SDK 版本</span><span class="sxs-lookup"><span data-stu-id="b03f6-110">Update .NET Core SDK version in global.json</span></span>

<span data-ttu-id="b03f6-111">如果您的方案依賴[global.json](/dotnet/core/tools/global-json)檔案，以特定的.NET Core SDK 版本為目標，更新其`version`屬性設為在電腦上安裝的 2.2 版本：</span><span class="sxs-lookup"><span data-stu-id="b03f6-111">If your solution relies upon a [global.json](/dotnet/core/tools/global-json) file to target a specific .NET Core SDK version, update its `version` property to the 2.2 version installed on your machine:</span></span>

```json

  "sdk": {
    "version": "2.2.100"
  }
}
```
