## <a name="c-tooling-support-for-proto-files"></a><span data-ttu-id="4bba2-116">C#適用于 proto 檔案的工具支援</span><span class="sxs-lookup"><span data-stu-id="4bba2-116">C# Tooling support for .proto files</span></span>

<span data-ttu-id="4bba2-117">gRPC 會使用合約優先的方法來開發 API。</span><span class="sxs-lookup"><span data-stu-id="4bba2-117">gRPC uses a contract-first approach to API development.</span></span> <span data-ttu-id="4bba2-118">服務和訊息會定義在 *\*的 proto*檔案中：</span><span class="sxs-lookup"><span data-stu-id="4bba2-118">Services and messages are defined in *\*.proto* files:</span></span>

```protobuf
syntax = "proto3";

service Greeter {
  rpc SayHello (HelloRequest) returns (HelloReply);
}

message HelloRequest {
  string name = 1;
}

message HelloReply {
  string message = 1;
}
```

<span data-ttu-id="4bba2-119">服務、用戶端和訊息的 .NET 類型會藉由在專案中包含 *\*的 proto*檔案來自動產生：</span><span class="sxs-lookup"><span data-stu-id="4bba2-119">.NET types for services, clients and messages are automatically generated by including *\*.proto* files in a project:</span></span>

```xml

  <Protobuf Include="Protos\greet.proto" />
</ItemGroup>
```

## <a name="grpc-services-on-aspnet-core"></a><span data-ttu-id="4bba2-123">ASP.NET Core 上的 gRPC 服務</span><span class="sxs-lookup"><span data-stu-id="4bba2-123">gRPC services on ASP.NET Core</span></span>

<span data-ttu-id="4bba2-124">gRPC 服務可以裝載于 ASP.NET Core 上。</span><span class="sxs-lookup"><span data-stu-id="4bba2-124">gRPC services can be hosted on ASP.NET Core.</span></span> <span data-ttu-id="4bba2-125">服務具有與熱門 ASP.NET Core 功能的完整整合，例如記錄、相依性插入（DI）、驗證和授權。</span><span class="sxs-lookup"><span data-stu-id="4bba2-125">Services have full integration with popular ASP.NET Core features such as logging, dependency injection (DI), authentication and authorization.</span></span>

<span data-ttu-id="4bba2-126">GRPC 服務專案範本提供入門服務：</span><span class="sxs-lookup"><span data-stu-id="4bba2-126">The gRPC service project template provides a starter service:</span></span>

```csharp
public class GreeterService : Greeter.GreeterBase
{
    private readonly ILogger<GreeterService> _logger;

    public GreeterService(ILogger<GreeterService> logger)
    {
        _logger = logger;
    }

    public override Task<HelloReply> SayHello(HelloRequest request,
        ServerCallContext context)
    {
        _logger.LogInformation("Saying hello to {Name}", request.Name);
        return Task.FromResult(new HelloReply 
        {
            Message = "Hello " + request.Name
        });
    }
}
```

<span data-ttu-id="4bba2-127">`GreeterService` 繼承自 `GreeterBase` 類型，這是從 *\** 的 `Greeter` 服務中產生的。</span><span class="sxs-lookup"><span data-stu-id="4bba2-127">`GreeterService` inherits from the `GreeterBase` type, which is generated from the `Greeter` service in the *\*.proto* file.</span></span> <span data-ttu-id="4bba2-128">此服務可供*Startup.cs*中的用戶端存取：</span><span class="sxs-lookup"><span data-stu-id="4bba2-128">The service is made accessible to clients in *Startup.cs*:</span></span>

```csharp
app.UseEndpoints(endpoints =>
{
    endpoints.MapGrpcService<GreeterService>();
});
```

<span data-ttu-id="4bba2-129">若要深入瞭解 ASP.NET Core 上的 gRPC 服務，請參閱 <xref:grpc/aspnetcore>。</span><span class="sxs-lookup"><span data-stu-id="4bba2-129">To learn more about gRPC services on ASP.NET Core, see <xref:grpc/aspnetcore>.</span></span>

## <a name="call-grpc-services-with-a-net-client"></a><span data-ttu-id="4bba2-130">使用 .NET 用戶端呼叫 gRPC 服務</span><span class="sxs-lookup"><span data-stu-id="4bba2-130">Call gRPC services with a .NET client</span></span>

<span data-ttu-id="4bba2-131">gRPC 用戶端是[從 *\*的 proto*檔案產生](xref:grpc/basics#generated-c-assets)的具體用戶端類型。</span><span class="sxs-lookup"><span data-stu-id="4bba2-131">gRPC clients are concrete client types that are [generated from *\*.proto* files](xref:grpc/basics#generated-c-assets).</span></span> <span data-ttu-id="4bba2-132">具體 gRPC 用戶端的方法會轉譯為 *\*的 proto*檔案中的 gRPC 服務。</span><span class="sxs-lookup"><span data-stu-id="4bba2-132">The concrete gRPC client has methods that translate to the gRPC service in the *\*.proto* file.</span></span>

```csharp
var channel = GrpcChannel.ForAddress("https://localhost:5001");
var client = new Greeter.GreeterClient(channel);

var response = await client.SayHelloAsync(
    new HelloRequest { Name = "World" });

Console.WriteLine(response.Message);
```

<span data-ttu-id="4bba2-133">GRPC 用戶端是使用通道所建立，這代表 gRPC 服務的長時間連接。</span><span class="sxs-lookup"><span data-stu-id="4bba2-133">A gRPC client is created using a channel, which represents a long-lived connection to a gRPC service.</span></span> <span data-ttu-id="4bba2-134">您可以使用 `GrpcChannel.ForAddress`來建立通道。</span><span class="sxs-lookup"><span data-stu-id="4bba2-134">A channel can be created using `GrpcChannel.ForAddress`.</span></span>

<span data-ttu-id="4bba2-135">如需建立用戶端和呼叫不同服務方法的詳細資訊，請參閱 <xref:grpc/client>。</span><span class="sxs-lookup"><span data-stu-id="4bba2-135">For more information on creating clients, and calling different service methods, see <xref:grpc/client>.</span></span>
