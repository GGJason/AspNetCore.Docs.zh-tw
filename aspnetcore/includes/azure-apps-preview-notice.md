> [!IMPORTANT]
> **使用 Azure App Service ASP.NET Core 3。0**
>
> ASP.NET Core 3.0 尚未部署至 Azure App Service。 我們希望很快就會提供 ASP.NET Core 3 Azure App Service。 若要裝載使用 ASP.NET Core 3.0 的應用程式：

* 將 ASP.NET Core 3.0 視為 Azure App Service 部署的預覽版本。
* 請參閱[將 ASP.NET Core 預覽版本部署至 Azure App Service](xref:host-and-deploy/azure-apps/index#deploy-aspnet-core-preview-release-to-azure-app-service)。
