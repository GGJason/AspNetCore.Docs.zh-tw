## <a name="cookies"></a><span data-ttu-id="83b9d-129">Cookie</span><span class="sxs-lookup"><span data-stu-id="83b9d-129">Cookies</span></span>

<span data-ttu-id="83b9d-130">Cookie 會在要求之間儲存資料。</span><span class="sxs-lookup"><span data-stu-id="83b9d-130">Cookies store data across requests.</span></span> <span data-ttu-id="83b9d-131">因為 Cookie 會隨著每個要求傳送，所以其大小應該保持最小。</span><span class="sxs-lookup"><span data-stu-id="83b9d-131">Because cookies are sent with every request, their size should be kept to a minimum.</span></span> <span data-ttu-id="83b9d-132">在理想情況下，應該只有識別碼儲存在 Cookie 中，而資料由應用程式儲存。</span><span class="sxs-lookup"><span data-stu-id="83b9d-132">Ideally, only an identifier should be stored in a cookie with the data stored by the app.</span></span> <span data-ttu-id="83b9d-133">大部分的瀏覽器將 Cookie 大小限制為 4096 個位元組。</span><span class="sxs-lookup"><span data-stu-id="83b9d-133">Most browsers restrict cookie size to 4096 bytes.</span></span> <span data-ttu-id="83b9d-134">每個網域只有數量有限的 Cookie 可供使用。</span><span class="sxs-lookup"><span data-stu-id="83b9d-134">Only a limited number of cookies are available for each domain.</span></span>
