<span data-ttu-id="39045-120">SignalR 自動處理連接管理，並像聊天室的同時，讓您廣播給所有連接的用戶端的訊息。</span><span class="sxs-lookup"><span data-stu-id="39045-120">SignalR handles connection management automatically, and lets you broadcast messages to all connected clients simultaneously, like a chat room.</span></span> <span data-ttu-id="39045-121">您也可以傳送訊息給特定的用戶端。</span><span class="sxs-lookup"><span data-stu-id="39045-121">You can also send messages to specific clients.</span></span> <span data-ttu-id="39045-122">用戶端與伺服器之間的連線是永久性的不同於傳統的 HTTP 連線，也就是重新建立每個通訊的。</span><span class="sxs-lookup"><span data-stu-id="39045-122">The connection between the client and server is persistent, unlike a classic HTTP connection, which is re-established for each communication.</span></span>

## <a name="transports-and-fallbacks"></a><span data-ttu-id="39045-132">傳輸和後援</span><span class="sxs-lookup"><span data-stu-id="39045-132">Transports and fallbacks</span></span>
