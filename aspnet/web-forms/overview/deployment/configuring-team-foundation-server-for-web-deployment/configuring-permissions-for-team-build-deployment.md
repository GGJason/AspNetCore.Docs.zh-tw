## <a name="configuring-web-server-permissions"></a><span data-ttu-id="21859-119">設定 Web 伺服器權限</span><span class="sxs-lookup"><span data-stu-id="21859-119">Configuring Web Server Permissions</span></span>

<span data-ttu-id="21859-120">中所述[選擇 Web 部署的權限方法](../configuring-server-environments-for-web-deployment/choosing-the-right-approach-to-web-deployment.md)，有兩種主要方法，如果您想要將 web 套件部署到遠端網頁伺服器，您可以使用：</span><span class="sxs-lookup"><span data-stu-id="21859-120">As described in [Choosing the Right Approach to Web Deployment](../configuring-server-environments-for-web-deployment/choosing-the-right-approach-to-web-deployment.md), there are two main approaches you can use if you want to deploy web packages to a remote web server:</span></span>

- <span data-ttu-id="21859-121">從遠端位置的應用程式部署將目標設為*Web Deployment Agent Service* （也稱為遠端代理程式） 在目的地伺服器上。</span><span class="sxs-lookup"><span data-stu-id="21859-121">Deploy the application from a remote location by targeting the *Web Deployment Agent Service* (also known as the remote agent) on the destination server.</span></span>
- <span data-ttu-id="21859-122">從遠端位置的應用程式部署將目標設為*Internet Information Services* (*IIS) Web 部署的處理常式*目的地伺服器上。</span><span class="sxs-lookup"><span data-stu-id="21859-122">Deploy the application from a remote location by targeting the *Internet Information Services* (*IIS) Web Deploy Handler* on the destination server.</span></span>

<span data-ttu-id="21859-123">遠端代理程式會在此情況下有兩個索引鍵的限制：</span><span class="sxs-lookup"><span data-stu-id="21859-123">The remote agent has two key limitations in this case:</span></span>

<span data-ttu-id="21859-129">相反地，Web 部署的處理常式方法會提供各種優點：</span><span class="sxs-lookup"><span data-stu-id="21859-129">In contrast, the Web Deploy Handler approach offers various advantages:</span></span>
